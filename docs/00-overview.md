# 项目总览

## 项目背景
spring-boot-iot 面向物联网网关平台建设，提供从设备接入到平台处理的统一工程骨架。

## 一期目标
先完成一个可运行的模块化单体版本，重点打通设备主链路：
- 产品定义
- 设备注册
- HTTP 模拟上报
- 协议解析
- 消息日志落库
- 最新属性更新
- 在线状态维护

## 一期范围内
- Maven 多模块工程骨架
- 基础配置与启动
- 核心数据库表
- 基础 API
- 协议适配器抽象
- HTTP 模拟设备上报

## 一期范围外
- 复杂规则引擎
- 完整告警通知
- 大规模网关边缘协同
- 全量 OTA 管理

## 二期规划
- MQTT 真接入
- 网关与子设备拓扑
- 设备影子
- 时序查询
- 指令记录

## 三期规划
- 规则引擎
- 告警中心
- OTA
- Open API
- 运维监控
