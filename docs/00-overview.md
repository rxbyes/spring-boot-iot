# spring-boot-iot 项目总览

## 项目背景
spring-boot-iot 是一个基于 Spring Boot 的物联网网关平台，面向设备接入、协议适配、遥测数据处理、设备管理与平台化扩展场景。

## 建设目标
- 提供统一的设备接入平台
- 支持多协议扩展
- 提供统一物模型
- 支持设备属性、消息日志、在线状态维护
- 为后续规则引擎、告警、OTA、网关拓扑奠定基础

## 一期范围
- 多模块 Maven 工程骨架
- 产品管理
- 设备管理
- HTTP 模拟设备上报
- 协议解析框架
- 消息日志落库
- 最新属性更新
- 设备在线状态更新

## 二期规划
- MQTT 真接入
- 网关与子设备管理
- 设备影子
- 历史时序查询
- 指令下发记录

## 三期规划
- 规则引擎
- 告警中心
- OTA 升级
- 多租户细化
- 开放 API
- 运维监控
